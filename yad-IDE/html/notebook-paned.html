yad --help-notebook

Usage:
  yad --notebook  [OPTION…] - Yet another dialoging program

Notebook options
  --notebook                                     Display notebook dialog
  --tab=LABEL                                    Add a tab to notebook
  --tab-pos=TYPE                                 Set position of a notebook tabs (top, bottom, left or right)
  --tab-borders=NUMBER                           Set tab borders
  --active-tab=NUMBER                            Set active tab
  --expand                                       Expand tabs
  --stack                                        Use stack mode

** Note: yad --notebook can have as many --tab="1" as memory in computer will allow.
** Note: yad --notebook must use random key. see below example


Example Script:
#!/ bin/bash
random key here 
yad --plug="$fkey" --tabnum="1" --html --browser -uri="https://google.com"
yad --plug="$fkey" --tabnum="2" --text-info  --wrap --filename="/home/linux/my-text-file.txt"
yad --notebook --key=$fkey --width=600 --height=600 --button="Close:1"

yad --help-paned

Usage:  Must use Randome Key:
  yad --paned [OPTION…] - Yet another dialoging program

Paned dialog options
  --paned                                        Display paned dialog
  --orient=TYPE                                  Set orientation (hor[izontal] or vert[ical])
  --splitter=POS                                 Set initial splitter position
  --focused=PANE  

** Note yad --paned can only have 2 --tabnum="1" --tabnum="2"
#!/ bin/bash
random key here
yad --plug="$fkey" --tabnum="1" --html --browser -uri="https://google.com"
yad --plug="$fkey" --tabnum="2" --text-info  --wrap --filename="/home/linux/my-text-file.txt"
yad --paned --key=$fkey --spitter=100 --width=600 --height=600 --button="Close:1"

Both yad --notebook & yad --paned must use a random key

------- Random Key Generator ------------
# Safe key generator for YAD (16-bit unsigned integer: 0 to 65535)
generate_yad_key() {
    echo $(( RANDOM % 65536 ))
}

# Generate keys for both containers
fkey=$(generate_yad_key)
fkey2=$(generate_yad_key)

# Ensure keys are different
while [ "$fkey2" -eq "$fkey" ]; do
    fkey2=$(generate_yad_key)
done
