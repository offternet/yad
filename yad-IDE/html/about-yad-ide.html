This is an Alpha Design Version 1.0x of yad IDE. We are working to add the General Snippets here and to the rest of the above tabs. Its actually easy to add your own. The name used for images must be exactly same as code snippet file. Images have *.png extension. Snippet code files do NOT have an extension.  image filename: color.png | snippet filename: color

Create Image:
1. Open one of the *.png images found in /production/yad/buttons into your image program. This will give you the correct size.

2. Add snippet name color to your image button. color is yad selection dialog.
   Flatten image (merge layers) and save as /production/yad/buttons/color.png

Create Snippet File:
3. Open a text editor and create the code snippet file for dialog: color. Snippet code for dialog color is: yad --color \
   We place yad in front of all dialogs (color is a yad dialog), yad  --color \
   Save file as /production/yad/color
   We don't use extenstions on snippet files in yad IDE.
  
   Use of backslash \
   The backslash \ is in most but, not all snippet files. The \ is for newline.
   Using \ Newline makes the code more readable as each snippet / command is on a new line.
   Later you can change to inline coding as you wish in this fashion:
     yad --html --browser --uri="https://yahoo.com" --width=1000 --height=600 --button="Exit:1")
